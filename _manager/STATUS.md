- [x] `wtPickVoice()` тепер ранжує кандидатів: точний regional-збіг (`en-US` vs просто `en`) +4, хмарний голос (`!localService`) +3, "Natural"/"Neural"/"Online" у назві +2, системний default +1 — і обирає найвищий скор
- [x] Той самий voice picker застосований і в Text Mode (`speakCurrentBlock`), а не лише в Words Mode — раніше там взагалі не було явного вибору голосу
